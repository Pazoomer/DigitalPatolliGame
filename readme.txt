# Digital Patolli Game

Multiplayer digital board game inspired by Patolli and games such as Sorry, developed in Java using a client-server architecture.

## Features
- Multiplayer gameplay
- Client-server system
- Thread-based player handling
- Digital board game mechanics
- Turn-based interactions

## Architecture
The project includes:
- A server application that manages the game logic and player connections
- Client applications used by players to join and play matches

## Technologies
- Java
- NetBeans
- Multithreading
- Socket communication

## Purpose
This project was developed to practice networking, multiplayer systems, thread management, and distributed game logic using Java.
